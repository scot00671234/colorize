# Colorizer 90-Day AEO + SEO Editorial Calendar

This calendar is aligned to current product truth:

- Web app focused on AI photo colorization
- Workspace upload -> colorize -> preview -> export

## Cluster Map

### Cluster A: Getting Started (high intent)
- Goal: convert new visitors who already want to colorize photos
- Primary CTA: `Create account` or `Open workspace`

### Cluster B: Input Quality & Troubleshooting
- Goal: improve activation and retention by helping users get better output
- Primary CTA: `Open workspace`

### Cluster C: Pricing & Plan Fit
- Goal: reduce plan confusion and support upgrades
- Primary CTA: `Compare plans`
