# Colorizer AEO + SEO Measurement and Quality Checklist

## Weekly KPI Review

Review every Monday and compare week-over-week:

- Search impressions by page (`/`, `/blog`, each `/blog/:slug`)
- Clicks by page and query group
- Average position for top 20 target queries
- Blog -> conversion assist:
  - blog visit -> register
  - blog visit -> pricing section
  - blog visit -> workspace open
- Scroll depth and CTA click-through on article pages

## Query Cluster Tracking

Track performance by cluster, not only by individual page:

- Getting Started
- Input Quality & Troubleshooting
- Pricing & Plan Fit

For each cluster log:

- total impressions
- total clicks
- CTR
- assisted conversions
- top 10 rising queries

## AEO Readiness Checklist (per page)

- Answer-first intro appears above the fold
- Primary question is explicit and human-readable
- FAQ section exists and is concise
- JSON-LD present and valid (Article + FAQ where applicable)
- Internal links to 2-4 related pages
- One CTA mapped to the page intent

## Product-Truth Checklist (must pass before publish)

- Feature claims match current app behavior
- Plan prices and limits match backend config
- No unsupported promises (for example guaranteed historical accuracy)
- Privacy/storage statements match published policy text

## Monthly Review Loop

- Refresh top 5 pages by impressions with stronger quick answers
- Expand FAQs for queries that trigger impressions but low CTR
- Replace weak CTAs where click-through is below baseline
- Prune or consolidate overlapping articles in same intent cluster
